Vocabulary

Internet
The Internet is a network of connected machines that send and recieve data to and from each other
TCP/IP
TCP/IP stand for Transmission Control Protocol/Internet Protocol which allows multiple computers to communitcate.
LAN
LAN stands for Local-Area Network and it is a computer network that other computers can only connect to in a limited area
IP address
An IP address is a destination in which a website is in on the Internet
HTTP
HTTP stands for Hypertext Transfer Protocol which is the language that computers send to each other to acess and display websites
web server
A web server is a piece of software that lets the computer connect to the World Wide Web and process several different network protocols
web client
The Web Client is also called the web browser which can have multiple other applications and plug ins
